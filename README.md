# Mila Nails — Static Demo Landing (HTML/CSS/JS)

A fully static, mobile-first nail tech landing page:
- Sticky header + anchor navigation
- EN/RU language switch (no reload, saved in localStorage)
- Portfolio grid built from `gallery.json`
- Filters + accessible lightbox (ESC close, arrows, focus trap)
- Contact form with basic validation and `mailto:` submit (no backend)

No external libraries. Works on any static hosting: Vercel / Netlify / GitHub Pages / обычный хостинг.
